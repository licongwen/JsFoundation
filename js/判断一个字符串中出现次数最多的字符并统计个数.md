## 判断一个字符串中出现次数最多的字符，统计这个次数。

> 基本思路：先将字符串变为数组，然后排序，将重复的个数放进一个数组中。返回数组中最大的值；

代码实现如下：
```js
function getMostStr(val){
    let arr = val.split('').sort();
    let j=0,arr1=[];
    for(let i=0;i<arr.length;i++){
        if(arr[i]==arr[i+1]){
            j++
        }else{
            arr1.push(j+1);
            j=0;   
        }
    }
    return Math.max(...arr1);
}
//测试用例
let str1 ='llllwwwddq';
let maxNum = getMostStr(str1);
console.log(maxNum)
```
